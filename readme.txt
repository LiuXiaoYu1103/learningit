1.配置用户名，邮箱
  git config --global user.name/user.email

2.初始化仓库
  git init

3.查看文件状态
  git status

4.添加文件到暂存区
  git add */-A

5.提交文件到仓库当中
  git commit -m"备注信息"

6.查看所有版本信息
  git log
  每一次提交都会对应一个sha:589738e18edf818142854a50c3efe82c5cf34a1b 
  该 版本是谁提交，提交日期。
